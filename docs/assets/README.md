# Diagram assets

This directory contains deterministic Mermaid sources for architecture diagrams used by the root README and docs.

Rules:

- Prefer Mermaid for documentation diagrams.
- Do not commit generated PNG diagrams for architecture docs unless there is a strong reason.
- Keep each maintained diagram as an editable Mermaid source.
- If a diagram is generated by an AI image model, keep the prompt next to the image. Current diagrams are
  hand-authored Mermaid, so there are no prompt files.
- Render diagrams in Markdown with fenced `mermaid` blocks when possible; keep this directory as the editable
  source catalog.
